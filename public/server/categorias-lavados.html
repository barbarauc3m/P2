Categorías de Lavado

Lavado Rápido

Ideal para ropa poco sucia o uso diario.

  • Temperatura: 30ºC
  • Duración: 30 min
  • Centrifugado: 1200 rpm
  • Detergente: 50 ml

Lavado Frío

Ideal para lana, seda, ropa interior, etc.

  • Temperatura: 20ºC
  • Duración: 40 min
  • Centrifugado: 600 rpm
  • Detergente: 30 ml

Lavado Caliente

Ideal para ropa muy sucia o con manchas.

  • Temperatura: 60ºC
  • Duración: 1h 15 min
  • Centrifugado: 1200 rpm
  • Detergente: 60 ml

Lavado Eco

Ideal para ahorrar energía y agua.

  • Temperatura: 40ºC
  • Duración: 1h
  • Centrifugado: 800 rpm
  • Detergente: 35 ml

Lavado Ropa Blanca

Ideal para camisas, toallas, sábanas, ropa blanca, etc.

  • Temperatura: 60ºC
  • Duración: 1h
  • Centrifugado: 1200 rpm
  • Detergente: 55 ml

Lavado Ropa Oscura

Ideal para vaqueros, camisetas negras, sudaderas, etc.

  • Temperatura: 40ºC
  • Duración: 1h
  • Centrifugado: 800 rpm
  • Detergente: 50 ml

Lavado Deporte

Ideal para ropa de deporte, poliester, lycra, etc.

  • Temperatura: 40ºC
  • Duración: 50 min
  • Centrifugado: 800 rpm
  • Detergente: 50 ml
